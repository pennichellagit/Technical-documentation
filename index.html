Introduction

JavaScript is a special language that computers understand. It's used to make websites more interesting and interactive.
You know when you click on buttons or see animations on a website? That's often done with JavaScript.

Imagine you're building a virtual Lego house on a website: JavaScript is like the magical tool you use to make the Lego bricks move, change colors, and even talk to you.
It helps you create fun stuff on the screen that responds when you do things like clicking or typing.

So, in short, JavaScript is the behind-the-scenes wizard that brings websites to life and makes them way more fun to use!

How to add JavaScript to an HTML

You add JavaScript to an HTML using the <script> tag. 

 
                <body>
                <script>

                console.log(“text”);

                </script>
                </body>

Some functions

The console is part of the web browser and allows you to log messages, run JavaScript code, and see errors and warnings.
Console.log is a function which generates output to the console.

JavaScript treats text and numbers differently: To use text, we have to enclose it in quotes: 

 
                console.log (“Very nice text”);
For numbers no quotes are needed: 
 
                console.log(42);
You can use this function as many times as you want. Each statement outputs text from a new line.

alert(); is another function which displays messages in an alert dialog box: 
 
                <script>

                function showAlert() {
                alert ("Surprise!");
                } 

                </script>
A computer program is a list of “instruction” to be “executed” by a computer. In a programming language, these instructions are called statements.

Variables

In JavaScript, a variable is a named container that stores data.
Variables are used to hold values of various types, such as numbers, strings, objects, arrays, and more.
They allow you to store and manipulate data in your code, in order to access them throughout the program.
You create them with the keyword let: 

 
                let example;

Initialization is when you give a variable a value. A value is assigned with =.


                let name;
                name="Sara";
Or, shorter: 
 
                let name="Sara";
After initializing a variable, we can output its value: 
 
                let name="Sara";
                console.log(name);
Variables can change values during the program: 
 
                let num=5;
                num=8;
                console.log(num);
When you run this code, it will assign the value 5 to the variable num, then reassign the value 8 to the same variable, and finally, it will log the value of num to the console. As a result, you will see 8 printed in the console output.

Naming rules for variables:
  • Must begin with a letter, an underscore _ or a dollar sign $;
  • It can't contain spaces;
  • Can contain only letters, numbers, underscores, dollar signs;
  • They are case-sensitive: Name and name are different.

Constants

const is the keyword for constants, which are variables who do not change. They must have a value when declared and it cannot change.
Using const means that the variable will never be assigned to a different value. 

 
                const color='red';

Data types

As we said, variables and constants hold values of specific data types.
Data type refers to the types of values a program can work with.

A string is a bunch of text or characters. You can create a string by putting text inside single quotes (' '), double quotes (" "), or backticks (` `).
Strings help you work with words, sentences, and other text-based stuff in your JavaScript programs.

Anything inside the quotes is treated as plain text, even if it looks like a number: 

 
                console.log("42");
The characters "4" and "2" are treated as two separate symbols within a string, not as a number.

Booleans are a data type that can only have TRUE or FALSE values. In numerical representations, 1 means TRUE and 0 is FALSE.

Comparison operators compare the values and results in Boolean values. Some of them are:

  • == equal to;
  • != not equal to;
  • > greater than;
  • < smaller than;
  • >= greater or equal to;
  • <= smaller or equal to;

=== is a strict equality comparison operator, it returns false for the values which are not a similar type. 

 
                let x=5;
                let y='5';
                console.log(x==y); // outputs TRUE
                console.log(x===y); // outputs FALSE, cause they are not the same data type

>< are also used to compare the string lexicographically, where the alphabetical order of words is based on the alphabetical order of their component letters. 

 
                console.log('a' < 'b'); // outputs TRUE
                console.log(“Bob” > “Dave”); // outputs FALSE

Conditional statements

'if', 'else if' and 'else' are conditional statements.

The if statement is used to run code in case the condition evaluates to TRUE.

 
                let purchase= 1700;
                if (purchase>=1500); {
                    console.log(“Discount!”); }

The condition is TRUE, so the code in the if statement will be executed.
If the condition is FALSE, then it won't be executed.
The condition is written in (), followed by {} where the code is written.

You can use any expression that can be evaluated as TRUE or FALSE for your if statement and they can be nested inside one other: the second if statement will only be performed if the condition of the first one is TRUE.

The else statement is used to run code in case the condition evaluates to FALSE.
It doesn't have its own condition, it just checks the condition of the if statement and runs the code if it is FALSE.

 
                let number= -5;
                if(number >= 0) {
                    console.log(“Positive number”); }
                else {
                    console.log(“Negative number”); }

else if is used to test multiple conditions and it's written between the if and else statements. You can use it many times. 

 
                let time= 15; 
                if(time < 10) {
                    console.log (“Good morning”); }
                else if(time < 20) {
                    console.log(“Good day”); }
                else {
                    console.log(“Good evening”); }
The program will move to the else if statements when the condition of the if statement is FALSE. If its condition is also FALSE, then the program will move to the else block.
